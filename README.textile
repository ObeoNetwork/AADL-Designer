h1. Welcome to AADL Designer by Obeo Network

This modeler provides a set of common diagrams to work with AADL models.
It is developped under the ESA Summer Of Code In Space 2012 program and in the context of the Polarsys IWG :
* http://sophia.estec.esa.int/socis2012/?q=about
* http://onad2012.blogspot.de/
* http://wiki.eclipse.org/Polarsys/ProjectsIdeas/SOCISObeo

It is based on Eclipse Sirius (www.eclipse.org/sirius).

These plugins are released under the EPL Open-Source License.

To build the module you'll need Maven3 and type @mvn clean verify checkstyle:checkstyle@
